Welcome

Welcome to the Vatsinator database service. It is where Vatsinator gets its data from. The Vatsinator database is open for everyone. It was created in order to always provide correct and updated data.

How does it work?

Vatsinator uses a couple of databases in order to work properly. For example, there is the airport database that lets the application show all the airports around the world. Of course, having more than 6000 airpots in the database there is a notifiable probability that some of them contain wrong coordinates, their name is mispelled or any other component is inaccurate. Whats is more, there are new airports being built and old ones are closed. That is why the Vatsinator online database service was created.

When a user launches Vatsinator, it synchronizes the local data with the remote server. The data set consists of:

  • the airport database - containing coordinates, name, location, ICAO and IATA codes and more;
  • the airline database - names, logos and links to the virtual equivalents.

Participate

The main goal of the Vatsinator database service is to provide accurate data. Therefore, this database can be freely updated and modified by anyone. We encourage everybody to modify any incorrect entries found. One can do that without any need to sign up or sign in.

Any commited changed has to be reviewed and approved by one of the moderators before it gets merged into the main database.
